# 🎈 RarinaEvent

This project is a custom website developed for **Rarina Event**, a creative event and family studio.  
The website presents the studio, upcoming events, gallery content, and pricing information in a visually friendly and welcoming way.

The project was built as a custom front-end website with **HTML**, **SCSS**, and **JavaScript**, and served with **Node.js + Express**.  
It also includes a **contact form with email sending functionality** using **Nodemailer**.
